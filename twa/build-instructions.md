# TWA APKビルド - 対話形式入力ガイド

## 重要な注意事項

⚠️ **`q`を入力しないでください！** `q`を入力するとコマンドが終了してしまいます。

## ビルドコマンド

```powershell
bubblewrap build
```

## 対話形式での入力手順

### ステップ1: 変更の適用確認

```
? There are changes in twa-manifest.json. Would you like to apply them to the project before building? (Y/n)
```

**入力**: **`Y`** と入力してEnter（または単にEnterキーを押す）

**注意**: 
- `n`を入力すると変更が適用されません
- **`q`を入力しないでください！**

### ステップ2: パスワード入力

パスワードの入力プロンプトが2回表示されます：

1. **Key Store Password**:
   ```
   ? Password for the Key Store: _
   ```
   **入力**: `P@ssword` と入力してEnter
   - 画面には表示されません（正常です）
   - ターミナルのカーソル位置に入力してください

2. **Key Password**:
   ```
   ? Password for the Key: _
   ```
   **入力**: `P@ssword` と入力してEnter
   - 画面には表示されません（正常です）

### ステップ3: ビルド実行

パスワード入力後、自動的にビルドが開始されます：
```
Building the Android App...
```

## よくある間違い

❌ **間違い**: `q`を入力 → コマンドが終了
✅ **正解**: `Y`を入力 → 変更が適用される

❌ **間違い**: パスワードを表示させようと繰り返し入力
✅ **正解**: パスワードは画面に表示されませんが、正しく入力されています

## トラブルシューティング

### `q`で終了してしまった

もう一度 `bubblewrap build` を実行してください。

### パスワードが受け付けられない

1. 正確に入力されているか確認（`P@ssword`）
2. Enterキーを押しているか確認
3. 大文字小文字を正確に入力

---

**次のアクション**: ターミナルで `bubblewrap build` を実行し、**`Y`**と入力してEnterを押してください（`q`は入力しないでください）。
