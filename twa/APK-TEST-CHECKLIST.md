## 🌐 Digital Asset Links検証

### 確認方法

1. **Androidデバイスで確認**:
   ```
   設定 → アプリ → まいにちAI姓名判断 → 開く → サイト設定
   ```
   - 「このサイトがアプリを検証済み」と表示されるか確認

2. **コマンドラインで確認**（開発者向け）:
   ```bash
   adb shell pm get-app-links com.nameanalysis.ai
   ```
   - `seimei.app`が表示されるか確認

3. **Webサイトで確認**:
   - `https://seimei.app/.well-known/assetlinks.json`がアクセス可能か確認
   - SHA256フィンガープリントが正しいか確認:
     ```
     B766698D95C2B3A1E236143DE6DC91343DFBFD5732C8C117F0F30E46F9DC15A9
     ```

## 🚨 トラブルシューティング

### APKがインストールできない場合
1. 「不明なソース」の設定を確認
2. ストレージ容量を確認
3. 別のデバイスで試す

### アプリが起動しない場合
1. ログを確認: `adb logcat | grep -i "nameanalysis"`
2. Digital Asset Linksが正しく設定されているか確認
3. Webサイト（`https://seimei.app/`）がアクセス可能か確認

### TWAモードで起動しない場合
1. Chromeが最新バージョンか確認
2. Digital Asset Linksの検証が完了しているか確認
3. `twa-manifest.json`の設定を確認
