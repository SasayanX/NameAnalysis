## 技術的な注意点

1. **フォント**: 
   - システムフォント優先（Noto Sans JP、Yu Minchoなど）
   - Webフォントは読み込みが遅いため、必要最小限に

2. **SVGフィルター**:
   - `feGaussianBlur`: 光輪・柔光
   - `feTurbulence`: 炎・雷光
   - `feMerge`: エフェクトの合成

3. **パターン生成**:
   - 龍の模様: SVGパスで描画（複雑な場合は簡略化）
   - 雲・炎: `feTurbulence`で生成
   - 木目・紙質: グラデーション + パターン

4. **パフォーマンス**:
   - SVGサイズの最適化
   - 不要なフィルターの削減
   - キャッシュの活用
