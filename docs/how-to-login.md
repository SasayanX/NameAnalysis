# ログイン方法ガイド

## ログインページへのアクセス

1. ブラウザで `http://localhost:3000/login` にアクセス
2. または、アプリ内のログインボタンからアクセス

## ログイン方法

### 方法1: メールアドレスとパスワードでログイン

1. `/login` ページにアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

### 方法2: 新規登録

1. `/login` ページにアクセス
2. 「アカウント作成はこちら」をクリック
3. 以下の情報を入力：
   - お名前（任意）
   - メールアドレス
   - パスワード
4. 「新規登録」ボタンをクリック
5. メール認証が有効な場合、メールボックスで認証リンクをクリック

### 方法3: Google認証（推奨）

1. `/login` ページにアクセス
2. 「Googleでログイン」ボタンをクリック
3. Googleアカウントで認証
4. 自動的にログイン完了

## デバッグ用KP付与ページでのログイン

1. `/debug-kp` ページにアクセス
2. ログインしていない場合、「ログインが必要です」というメッセージが表示されます
3. 「ログインページへ」リンクをクリック
4. 上記の方法でログイン
5. ログイン後、自動的に `/debug-kp` ページに戻ります

## 注意事項

- Supabase環境変数が設定されている必要があります
- 開発環境では、メール認証がスキップされる場合があります
- Google認証を使用する場合、SupabaseでGoogle OAuth設定が必要です

## トラブルシューティング

### 「Supabase環境変数が設定されていません」エラー

`.env.local` ファイルに以下が設定されているか確認：

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### ログインできない

1. メールアドレスとパスワードが正しいか確認
2. メール認証が必要な場合、メールボックスを確認
3. Google認証を使用してみる
4. Supabaseダッシュボードでユーザーが作成されているか確認
