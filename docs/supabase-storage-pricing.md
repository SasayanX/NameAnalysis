# Supabase Storage 料金・制限ガイド

## 無料プラン（Free Tier）

### ストレージ容量
- **1GBまで無料**で利用可能
- 画像ファイル（PNG）の平均サイズを100KBと仮定すると、約10,000枚のカードを保存可能

### 保存期間
- **無制限**（保存期間の制限なし）
- 一度アップロードしたファイルは削除しない限り永続的に保存されます

### その他の制限
- **帯域幅**: 月間5GB（ダウンロード/アップロード合計）
- **月間アクティブユーザー（MAU）**: 50,000人まで
- **データベース容量**: 500MB（Storageとは別）
- **プロジェクト数**: 最大2つ

### 自動停止の注意点
⚠️ **重要**: プロジェクトが**7日間アクセスされない**と、自動的に一時停止されます
- 一時停止後もデータは保持されます
- プロジェクトにアクセスすると自動的に再開されます
- 定期的にアクセスがある限り、問題ありません

## 有料プラン

### Proプラン（$25/月）
- **ストレージ容量**: 100GB
- **帯域幅**: 月間250GB
- **月間アクティブユーザー**: 無制限
- **自動停止**: なし

### Teamプラン（$599/月）
- **ストレージ容量**: 400GB
- **帯域幅**: 月間1TB
- その他の機能も大幅に拡張

## カード画像の容量見積もり

### 想定シナリオ
- **画像サイズ**: 約100-200KB/枚（1024x1536px PNG）
- **1GBあたり**: 約5,000-10,000枚のカード

### 無料プランでの運用
- 月に100枚のカードが発行される場合
- 1年で約1,200枚（約120-240MB）
- **1GBあれば約4-8年分**の容量が確保できます

## コスト最適化のヒント

1. **画像最適化**: PNGの圧縮率を調整（品質を少し下げる）
2. **古いカードの削除**: 一定期間（例：1年）経過したカードを自動削除
3. **ユーザー削除機能**: ユーザーが不要になったカードを削除できる機能を提供

## まとめ

✅ **無料プランで十分な場合**:
- 小規模なサービス（月100枚程度）
- 画像サイズを最適化している
- 定期的にアクセスがある（7日以内）

⚠️ **有料プランが必要な場合**:
- 大規模なサービス（月1,000枚以上）
- 高解像度画像が必要
- 自動停止を避けたい
