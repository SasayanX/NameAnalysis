# サーバーのログの確認方法

## 📋 サーバーのログとは？

**サーバーのログ** = 開発サーバー（`npm run dev`）を実行している**ターミナル（コマンドプロンプトやPowerShell）**に表示される出力のことです。

## 🔍 ログの確認方法

### Step 1: 開発サーバーが起動しているか確認

開発サーバーを起動しているターミナルウィンドウを開いてください。

通常、以下のような表示が出ています：

```
▲ Next.js 15.2.4
- Local:        http://localhost:3000
- Ready in 2.5s
```

### Step 2: オートパイロットを実行

ブラウザで `/data-expansion` にアクセスして、「オートパイロット実行」ボタンをクリックしてください。

### Step 3: ターミナルに表示されるログを確認

オートパイロットを実行すると、ターミナルに以下のようなログが表示されます：

```
🚀 オートパイロット実行開始
📅 実行時間: 2025-01-XX...
🔍 Twitter API認証情報チェック:
  - TWITTER_API_KEY: ✅ 設定済み
  - TWITTER_API_SECRET: ✅ 設定済み
  - TWITTER_ACCESS_TOKEN: ✅ 設定済み
  - TWITTER_ACCESS_TOKEN_SECRET: ✅ 設定済み
  - NODE_ENV: development

🐦 Xへの投稿開始: 大谷翔平さん
📝 ツイート内容: 🔮【大谷翔平さんの姓名判断】...
🔐 OAuth署名生成完了（v2エンドポイント）
📝 リクエストURL: https://api.twitter.com/2/tweets
📋 OAuth認証情報:
  - API Key: 0QdwZhPitm...
  - Access Token: 1918491750...
📤 リクエスト送信開始...
📡 レスポンスステータス: 403
❌ Twitter APIエラー詳細:
  - ステータス: 403
  - エラーデータ: {...}
```

## 📸 ログの見方

### 重要なログ

1. **環境変数の状態**
   ```
   🔍 Twitter API認証情報チェック:
     - TWITTER_API_KEY: ✅ 設定済み または ❌ 未設定
   ```

2. **エラーの詳細**
   ```
   ❌ Twitter APIエラー詳細:
     - ステータス: 403
     - エラーデータ: {...}
   ```

3. **OAuth署名の情報**
   ```
   🔐 OAuth署名生成完了（v2エンドポイント）
   📋 OAuth認証情報:
     - API Key: 0QdwZhPitm...
   ```

## 🖼️ ターミナルの場所

### Windowsの場合

1. **コマンドプロンプト**（cmd.exe）
2. **PowerShell**
3. **VS Codeの統合ターミナル**

### 確認方法

`npm run dev` を実行しているウィンドウを探してください。通常、以下のような表示があります：

```
> next dev

▲ Next.js 15.2.4
- Local:        http://localhost:3000
```

## 📋 ログをコピーする方法

### Windowsの場合

1. ターミナルで**右クリック** → **範囲指定**
2. 表示したいログ部分を**ドラッグして選択**
3. **Enterキー**を押すとクリップボードにコピーされます
4. メッセージに貼り付けてください

または：

1. ターミナルで**右クリック** → **すべて選択**
2. **Enterキー**を押してコピー
3. メッセージに貼り付け

## 💡 例

オートパイロット実行後、ターミナルに表示されるログを**コピーして貼り付けてください**。

特に以下の部分が重要です：
- エラーメッセージの詳細
- レスポンスステータス
- OAuth認証情報の状態

この情報があれば、問題の原因を特定できます！
