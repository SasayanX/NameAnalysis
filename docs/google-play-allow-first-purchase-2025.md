# Google Play Console - 初回購入許可設定（2025年最新版）

## エラーメッセージ

```
デベロッパーが初回購入を許可していません
```

## 重要な変更点

Google Play ConsoleのUIが変更され、「初回購入を許可する」設定の場所が変わった可能性があります。

## 解決方法（最新版）

### ステップ1: 商品がアクティブになっているか確認 ⭐ 最重要

1. [Google Play Console](https://play.google.com/console)にログイン
2. **まいにちAI姓名判断** アプリを選択
3. **収益化** → **商品** → **サブスクリプション** を選択
4. 商品の状態を確認：
   - **「アクティブ」** になっている必要があります
   - **「下書き」** の場合は、まずアクティブ化が必要です

### ステップ2: 商品をアクティブ化

1. 商品の詳細ページを開く
2. ページ上部の **「アクティブ」** または **「公開」** ボタンをクリック
3. 確認ダイアログで **「アクティブにする」** をクリック

### ステップ3: ライセンステストの設定を確認 ⭐ 重要

1. **設定** → **ライセンステスト** を選択
2. **ライセンステストを有効にする** がオンになっているか確認
3. **Gmailアカウント** にテスト用のGoogleアカウントが追加されているか確認
4. テスト用のGoogleアカウントを追加（まだ追加していない場合）
5. **保存** をクリック

**重要**: ライセンステストが有効になっていないと、購入テストができません。

### ステップ4: 商品編集ページで設定を確認

1. **収益化** → **商品** → **サブスクリプション** を選択
2. 商品をクリックして編集ページを開く
3. 以下のセクションを確認：

#### 確認すべき場所

1. **商品の基本情報セクション**
   - 商品名、説明の近く
   - **「初回購入を許可する」** または **「Allow first-time purchase」** のチェックボックス

2. **価格設定セクション**
   - 価格の設定の近く
   - **「初回購入を許可する」** のチェックボックス

3. **詳細設定セクション**
   - その他の設定項目と一緒に
   - **「初回購入を許可する」** のチェックボックス

4. **商品の状態セクション**
   - ページ上部の **「アクティブ」** トグルの近く
   - **「初回購入を許可する」** のチェックボックス

### ステップ5: 商品を再作成する（設定が見つからない場合）

「初回購入を許可する」設定が見つからない場合、商品を再作成する必要があるかもしれません：

1. **収益化** → **商品** → **サブスクリプション** を選択
2. **「商品を作成」** ボタンをクリック
3. 商品作成ウィザードで、以下の情報を入力：
   - **商品ID**: `basic_monthly` または `premium_monthly`
   - **名前**: ベーシックプラン または プレミアムプラン
   - **価格**: 330円 または 550円
   - **課金期間**: 1ヶ月
4. 商品作成ウィザードの各ステップで、**「初回購入を許可する」** のチェックボックスを探す
5. チェックボックスを**有効にする**
6. 商品作成を完了

## よくある問題と解決方法

### 問題1: 「初回購入を許可する」設定が見つからない

**解決方法**:
1. 商品が**「アクティブ」**になっているか確認
2. 商品編集ページ全体をスクロールして確認
3. 商品を再作成してみる

### 問題2: 設定してもエラーが出る

**解決方法**:
1. **ライセンステスト**が有効になっているか確認
2. テスト用のGoogleアカウントが追加されているか確認
3. 数分待ってから再度試す（設定の反映に時間がかかる場合がある）

### 問題3: 商品がアクティブ化できない

**解決方法**:
1. **マーチャントアカウント**が正しく設定されているか確認
2. 商品の必須項目がすべて入力されているか確認
3. 商品の価格が正しく設定されているか確認

## 確認チェックリスト

- [ ] 商品が**「アクティブ」**になっている
- [ ] **ライセンステスト**が有効になっている
- [ ] テスト用のGoogleアカウントが追加されている
- [ ] **「初回購入を許可する」**が有効になっている（設定が見つかる場合）
- [ ] マーチャントアカウントが設定されている

## 次のステップ

1. **商品をアクティブ化** - 商品がアクティブになっているか確認
2. **ライセンステストを有効化** - テスト用のGoogleアカウントを追加
3. **商品を再作成** - 設定が見つからない場合は再作成を試す

問題が続く場合は、Google Play Consoleのサポートにお問い合わせください。
