# Twitter APIプラン確認ガイド

## 🔍 現在のプランを確認

### Developer Portalで確認

1. [Twitter Developer Portal](https://developer.twitter.com/en/portal/dashboard)にログイン
2. トップページの**プロジェクト名の下**にプランが表示されている
3. または、**Settings** → **Subscription**で確認

表示されるプラン：
- **Free**: 無料プラン
- **Basic**: $100/月
- **Pro**: $5,000/月
- **Enterprise**: カスタム価格

## 📊 プラン別の制限

### Freeプラン

**理論上は**:
- OAuth 1.0aでv2エンドポイントにアクセス可能
- 月間1,500件のツイート投稿が可能

**実際には**:
- 一部のエンドポイントに制限がある可能性
- OAuth 1.0aでの書き込みが制限されている可能性

### Basicプラン（$100/月）

- より多くのエンドポイントにアクセス可能
- OAuth 1.0aでの書き込みが確実に動作
- レート制限が緩和

### Proプラン（$5,000/月）

- すべてのエンドポイントにアクセス可能
- 高いレート制限

## 🤔 現在の状況

設定は正しく完了しているのにエラーが続く場合：

### 可能性1: Freeプランでは制限がある

- FreeプランではOAuth 1.0aでの書き込みが制限されている可能性
- 有料プラン（Basic以上）が必要な可能性

### 可能性2: 設定反映の遅延

- 設定変更後、反映に時間がかかる（30分〜1時間以上）
- もう少し待ってから再試行

### 可能性3: アプリの状態

- アプリが「Suspended」や「Inactive」になっている可能性
- Developer Portalで確認

## 💡 推奨事項

### 1. プランを確認

Developer Portalで現在のプランを確認してください。

### 2. 有料プランが必要な場合

もしFreeプランで制限がある場合は：
- **Basicプラン（$100/月）**へのアップグレードを検討
- または、X投稿機能を一時的に無効化

### 3. 一時的な対応

有料プランへのアップグレードが難しい場合：
- X投稿機能を一時的に無効化
- オートパイロットの他の機能（データ拡充、メール送信）は継続
- 後で有料プランにアップグレード後にX投稿機能を有効化
